# OCT

![OCT](./images/OCT.png)

OCTは1V/octピッチ情報用のオクターブトランスポーズツールです。入力ノートを正確なオクターブ数だけ上下にシフトできます。

## コントロール

- **Octave：** オクターブトランスポーズ、-4（現在のピッチの4オクターブ下）から+4（4オクターブ上）まで

## ジャック

- **CV In：** オクターブパラメーター用CV入力、-5v/+5v
- **Input：** ノートCV信号
- **Out：** トランスポーズされたノートCV信号
