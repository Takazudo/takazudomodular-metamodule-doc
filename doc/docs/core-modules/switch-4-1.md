# Switch 4:1

![Switch 4:1](./images/Switch_4-1.png)

Switch 4:1は、4つの信号を受け取り1つの出力先にルーティングするシーケンシャルスイッチモジュールです。

## ジャック

- **In 1〜4：** ルーティングされる信号
- **Clock In：** スイッチを次のステップに進めるメインクロック入力
- **CV In：** ステッピングの代わりに4つの入力間をクロスフェードするCV入力、0〜5v
- **Reset In：** シーケンシャルスイッチをステップ1にリセットするゲート入力
- **Out：** スイッチの出力
