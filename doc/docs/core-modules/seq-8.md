# SEQ 8

![SEQ 8](./images/SEQ8.png)

SEQ8は基本的な8ステップCVシーケンサーです。

## コントロール

- **ノブ1〜8：** 0〜10vオフセット

## ジャック

- **Clock Gate In：** モジュールのメインクロック入力。ゲートまたはトリガーを受け入れます。
- **Reset Gate In：** リセット入力、Highの時にシーケンスをステップ0に設定
- **Gate Out：** シーケンスのステップ1でステップを発火
- **Out：** アクティブなステップのCV出力
