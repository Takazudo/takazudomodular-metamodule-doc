# CLKM

![CLKM](./images/CLKM.png)

CLKMは、入力ゲートを受け取り、そのタイミングを指定された量で逓倍するシンプルなクロックマルチプライヤーです。

## コントロール

- **Multiply：** "x1"（逓倍なし）から"x16"まで、ステップで設定

## ジャック

- **CV：** クロック逓倍用CV入力、-5v/+5v
- **Clk In：** クロック入力
- **Clock：** クロック出力
