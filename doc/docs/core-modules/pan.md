# Pan

![Pan](./images/Pan.png)

Panはステレオフィールド内で信号をパンニングするためのシンプルなユーティリティです。

## コントロール

- **Pan：** ステレオフィールド内でサウンドを左から右に移動

## ジャック

- **CV In：** パン用CV入力、-5v/+5v
- **Input：** オーディオ入力
- **Out 1：** 左オーディオ出力
- **Out 2：** 右オーディオ出力
