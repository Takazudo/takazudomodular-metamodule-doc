# CLKD

![CLKD](./images/CLKD.png)

CLKDは、入力ゲートを受け取り、そのタイミングを指定された量で分周するシンプルなクロックディバイダーです。

## コントロール

- **Divide：** "="（分周なし）から/16まで、ステップで設定

## ジャック

- **CV：** クロック分周用CV入力、-5v/+5v
- **Clk In：** クロック入力
- **Clock：** クロック出力
