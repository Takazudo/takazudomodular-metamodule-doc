# Verb

![Verb](./images/Verb.png)

追加コントロールを持つFreeverbアルゴリズムに基づいた実験的なリバーブモジュールです。しばしばノイジーで、ハーシュで、予測不能です。

## コントロール

- **Size：** 「部屋」のサイズ
- **Time：** リバーブのディケイタイム
- **Damp：** リバーブでディケイする高周波の量
- **AP Ratio：** オールパスフィルターサイズの比率
- **Comb：** コムフィルターサイズの比率
- **Mix：** リバーブ量のドライ/ウェットコントロール

## ジャック

- **Input：** オーディオ入力
- **Size CV In：** サイズ用CV入力、-5v/+5v
- **Time CV In：** タイム用CV入力、-5v/+5v
- **Damp CV In：** ダンプ用CV入力、-5v/+5v
- **Ratio CV In：** レシオ用CV入力、-5v/+5v
- **Comb CV In：** コム用CV入力、-5v/+5v
- **Mix CV In：** ミックス用CV入力、-5v/+5v
- **Out：** オーディオ出力
