# Complex EG

![Complex EG](./images/Complex_EG.png)

Complex EGはADSHRエンベロープジェネレーターで、アタック、ディケイ、リリースタイムのカーブ形状を制御できます。このエンベロープは、長いノートホールドで時間とともにサウンドの複雑な変化を制御できるキーボードで特に便利です。

## コントロール

- **Attack：** エンベロープのアタックフェーズ（立ち上がり時間）、1ms〜1秒
- **Decay：** エンベロープのディケイフェーズ（アタックからリリースステージまでの時間）、1ms〜1秒
- **Release：** エンベロープのリリースフェーズ（サスティンステージ後、信号が0vに落ちるまでの時間）、1ms〜1秒
- **A Curve：** アタックカーブのタイムスケーリングを指数関数（0%）からリニア（50%）から対数関数（100%）まで連続的に変更
- **D Curve：** ディケイカーブのタイムスケーリングを指数関数（0%）からリニア（50%）から対数関数（100%）まで連続的に変更
- **R Curve：** リリースカーブのタイムスケーリングを指数関数（0%）からリニア（50%）から対数関数（100%）まで連続的に変更
- **Sustain：** サスティンフェーズ（ディケイステージ後にサウンドが保持される音量）、0〜8v
- **Loop：** エンベロープのループ動作を有効にするトグル。オフの場合、エンベロープは外部ゲートが必要
- **Hold：** エンベロープのホールドフェーズ（入力ゲートがオフになってからリリースフェーズが始まるまでの時間）、1ms〜1秒

## ジャック

- **Gate In：** エンベロープジェネレーターを開始するゲート入力
- **Attack CV In：** アタックタイム用CV入力、-5v/+5v
- **Decay CV In：** ディケイタイム用CV入力、-5v/+5v
- **Release CV In：** リリースタイム用CV入力、-5v/+5v
- **Sustain CV In：** サスティン音量用CV入力、-5v/+5v
- **Hold CV In：** ホールドタイム用CV入力、-5v/+5v
- **Attack Out：** アタックステージがアクティブな時の8vロジック出力
- **Decay Out：** ディケイステージがアクティブな時の8vロジック出力
- **Release Out：** リリースステージがアクティブな時の8vロジック出力
- **Sustain Out：** サスティンステージがアクティブな時の8vロジック出力
- **Hold Out：** ホールドステージがアクティブな時の8vロジック出力
- **Envelope Out：** フルエンベロープ出力
