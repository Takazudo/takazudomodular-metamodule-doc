# HPF

![HPF](./images/HPF.png)

HPFは12dB/Octのレゾナント・ハイパスフィルターで、2つのモードがあります：「Standard」とMS20のKorg Sallen-Keyスタイルのハイパスをミミックをした「Korg」です。

## コントロール

- **Cutoff：** フィルターカットオフ、130hz〜2093hz
- **Q：** フィルターレゾナンス、1x〜20x
- **Mode：** ハイパスフィルタースタイル、Standard（左）またはKorg（右）

## ジャック

- **CV Input：** フィルターカットオフ用CV入力、-5v/+5v
- **Input：** オーディオ入力
- **Out：** オーディオ出力
