# Djembe

![Djembe](./images/Djembe.png)

Djembeは、実際のアコースティック・ジャンベの音響特性に基づいたドラムモジュールです。シンプルなエキサイターとレゾナントフィルターを使用しています。

## コントロール

- **Pitch：** ドラムの全体的なピッチ、20hz〜500hz
- **Sharpness：** ドラムの全体的な音量
- **Hit：** サウンドのエキサイター部分用ローパスフィルター
- **Strike Amount：** エキサイター音量

## ジャック

- **Pitch CV In：** ピッチCV入力、-5v/+5v
- **Sharp CV In：** シャープCV入力、-5v/+5v
- **Hit CV In：** ヒットCV入力、-5v/+5v
- **Strike CV In：** ストライクCV入力、-5v/+5v
- **Trigger In：** トリガー入力
- **Out：** オーディオ出力
