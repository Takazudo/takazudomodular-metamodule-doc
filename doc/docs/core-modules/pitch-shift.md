# Pitch Shift

![Pitch Shift](./images/Pitch_Shift.png)

Pitch Shiftはシンプルなピッチシフターです。

## コントロール

- **Coarse：** セミトーンシフト、-12〜+12セミトーン
- **Fine：** ピッチシフトの微調整、-1〜1セミトーン
- **Window：** バッファサイズ、0.42ms〜200ms
- **Mix：** エフェクトのドライ/ウェットクロスフェーダー

## ジャック

- **Input：** オーディオ入力
- **Pitch CV In：** ピッチ用CV入力、-5v/+5v
- **Window CV In：** ウィンドウ用CV入力、-5v/+5v
- **Mix CV In：** ミックス用CV入力、-5v/+5v
- **Out：** オーディオ出力
