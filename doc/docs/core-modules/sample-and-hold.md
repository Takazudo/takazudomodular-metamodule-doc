# S&H

![S&H](./images/S_H.png)

S&Hはシンプルなデュアル・サンプル＆ホールドモジュールです。

## ジャック

- **Sam：** サンプル＆ホールドされるチャンネル1のソース入力
- **In 1：** サンプル＆ホールドを開始するチャンネル1のクロックゲート入力
- **Samp：** サンプル＆ホールドされるチャンネル2のソース入力
- **In 2：** サンプル＆ホールドを開始するチャンネル2のクロックゲート入力
- **Out 1：** チャンネル1の出力
- **Out 2：** チャンネル2の出力
