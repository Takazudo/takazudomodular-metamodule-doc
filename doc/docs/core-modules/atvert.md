# Atvert

![Atvert](./images/Atvert.png)

Atvertは2チャンネルのアッテヌバーターで、2つのノブを使って2つの入力信号をスケーリングおよび反転できます。チャンネルに入力が接続されていない場合は、+5Vのオフセットが入力として使用されます（結果として出力は-5V〜+5Vになります）。

## コントロール

- **ノブ（各チャンネルに1つ）：** -100%（反転）から+100%（通常）まで調整。中央=0%（出力なし）

## ジャック

- **Input：** CV入力
- **Out：** アッテヌエート/反転された出力
