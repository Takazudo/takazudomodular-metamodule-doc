# FLW

![FLW](./images/FLW.png)

FLWは、オーディオまたはCV信号の振幅に基づいて指数/対数エンベロープを生成するエンベロープフォロワーです。さらに、コンパレーター出力があります。

## コントロール

- **Rise：** エンベロープの立ち上がり時間、1ms〜2秒。立ち上がりは非線形で、最初は速く、その後遅くなります。
- **Fall：** エンベロープの立ち下がり時間、1ms〜2秒。立ち下がりは非線形で、最初は速く、その後遅くなります。
- **Thresh：** ゲートが出力される電圧閾値。エンベロープ信号がこの電圧より高い場合、ゲートはHigh。低い場合はLow。

## ジャック

- **Input：** CVまたはオーディオ入力
- **Gate Out：** エンベロープが閾値を超えている時にHighになるゲート出力
- **Env Out：** フォロワーのエンベロープ出力
