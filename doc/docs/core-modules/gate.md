# Gate

![Gate](./images/Gate.png)

Gateはゲート信号のモディファイアです。可変パルス幅とディレイを持つワンショットパルスジェネレーターがあります。

## コントロール

- **Length：** ゲートのオン時間を調整、1ms〜1秒
- **Delay：** 入力ゲートと出力ゲートの間のディレイタイム、0ms〜1秒

## ジャック

- **Length CV In：** レングスCV入力、-5v/+5v
- **Delay CV In：** ディレイCV入力、-5v/+5v
- **Input：** ゲート入力
- **Out：** ゲート出力
