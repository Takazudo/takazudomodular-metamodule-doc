# Freeverb

![Freeverb](./images/Freeverb.png)

Freeverbは、人気のあるパブリックドメインのFreeverbアルゴリズムに基づいた軽量リバーブエフェクトです。オールパスフィルターとコムフィルターのコレクションを使用しています。

## コントロール

- **Size：** 部屋のサイズ（ディレイの長さ）
- **Feedback：** リバーブ信号が自身にフィードバックされる量
- **Damp：** リバーブ信号のローパスフィルタリング
- **Mix：** リバーブミックスのドライ/ウェット

## ジャック

- **Input：** オーディオ入力
- **Size CV In：** サイズ用CV入力、-5v/+5v
- **Feedback CV In：** フィードバック用CV入力、-5v/+5v
- **Damp CV In：** ダンプ用CV入力、-5v/+5v
- **Mix CV In：** ミックス用CV入力、-5v/+5v
- **Out：** オーディオ出力
