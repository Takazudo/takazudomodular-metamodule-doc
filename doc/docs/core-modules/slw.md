# SLW

![SLW](./images/Slew.png)

SLWは入力CVまたはオーディオを処理するためのシンプルなアタック/リリース・スルーリミッターです。

## コントロール

- **Rise：** スルーリミッターの立ち上がり時間、1ms〜2秒
- **Fall：** スルーリミッターの立ち下がり時間、1ms〜2秒

## ジャック

- **Input：** CVまたはオーディオ入力
- **Out：** スルー処理された電圧出力
