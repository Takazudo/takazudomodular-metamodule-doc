# Stereo Mixer

![Stereo Mixer](./images/Stereo_Mixer.png)

Stereo Mixerは、チャンネルごとにパンを持つ4チャンネルステレオオーディオミキサーです。

## コントロール

- **Level 1〜4：** 各チャンネルの音量
- **Pan 1〜4：** 各チャンネルのパン、ステレオフィールド内で信号を左または右にシフト

## ジャック

- **In L 1〜4：** チャンネルごとのオーディオ左入力
- **In R 1〜4：** チャンネルごとのオーディオ右入力
- **L Out：** ミックス出力サム左チャンネル
- **R Out：** ミックス出力サム右チャンネル
