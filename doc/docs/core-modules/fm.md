# FM

![FM](./images/FM.png)

FMは可変波形を持つ2オペレーターFMオシレーターです。

## コントロール

- **Pitch：** キャリアとモジュレーターのベース周波数、20hz〜20khz
- **Mix：** キャリアと変調されたキャリアの間をブレンドするクロスフェーダー
- **Index：** 周波数FMのデプス
- **Ratio C：** モジュレーターピッチ比の粗調整
- **Ratio F：** モジュレーターピッチ比の微調整
- **Shape：** サイン波からスクエア波まで連続的にブレンド

## ジャック

- **Index CV：** Indexパラメーター用アッテヌバーター + CV
- **Shape CV：** Shapeパラメーター用アッテヌバーター + CV
- **V/Oct C CV In：** キャリア用1v/Octピッチ入力
- **V/Oct M CV In：** モジュレーター用1v/Octピッチ入力
- **Mix CV In：** ミックス用CV入力、-5v/+5v
- **Index CV In：** インデックス用CV入力、-5v/+5v
- **Shape CV In：** シェイプ用CV入力、-5v/+5v
- **Out：** オーディオ出力
