# 技術仕様

## 概要

このドキュメントページでは、消費電力、物理寸法、接続オプションを含むMetaModuleシンセサイザーモジュールの技術仕様について説明します。

## 主な仕様

### 消費電力

モジュールは接続されているものによって異なる量の電力を消費します：

**+12Vレール：**

- 基本使用：「255mA標準、USBまたはmicroSDカード未接続時」
- microSDカードアクセス時：265mA
- USB 3.0ドライブ使用時：280mA
- 各種MIDIコントローラー使用時：275mA（Arturia MicroLab）から最大495mA（理論上の最大）

**その他のレール：**

- -12V：24mA
- +5V：使用しない

### 物理寸法

- **幅：** 26HP（132mm / 5.2インチ）
- **高さ：** 128.5mm（5.059インチ）
- **必要なクリアランス：** 電源ケーブル装着時、パネル下25mm

### 接続性

**USB-Cジャック：**

- USB 2.0および3.0マスストレージデバイスをサポート
- 最大500mA +5V電源供給でMIDIデバイス対応

**microSDカードスロット：**

- 標準スピードクラス（C2-C10、U1、U3）をサポート
- UHS-IおよびUHS-IIバスカードに対応
